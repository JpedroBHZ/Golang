# Go Mastery: Core Fundamentals & REST Development

This repository contains my progress through an intensive Go (Golang) specialization. It serves as my foundational technical base, covering the language's internals up to web service implementation.

## 🎓 Learning Path Summary

* **Concurrency Control**: Implementing parallel tasks using Goroutines and Channels (Folder 10).
* **Memory Management**: Practical application of Pointers and memory addressing (Folder 04).
* **Decoupled Design**: Utilizing Interfaces to write generic, testable code (Folder 06).
* **Data Structures**: Advanced manipulation of Slices, Maps, and Custom Structs (Folders 05 & 07).
* **Web Services**: Developing a RESTful API from scratch, handling HTTP methods and JSON (Folder 11).
